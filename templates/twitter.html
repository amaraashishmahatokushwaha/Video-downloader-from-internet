How to Use

How do I download a single tweet's content?
  1. Copy the URL of the tweet you want to download content from
  2. Make sure the "Tweet Content" tab is selected
  3. Paste the URL in the input field
  4. Choose your preferred format (Video HD, Video SD, GIF, or Images)
  5. Select any additional options you want
  6. Click "Start Download"
  7. Your files will be saved in the "downloads/twitter" folder
How do I download media from a user's profile?
  1. Find the Twitter username (handle) of the person
  2. Click on the "User Content" tab
  3. Enter the username with or without the @ symbol
  4. Choose what type of media you want to download (Videos, Images, GIFs, or All)
  5. Adjust the number of recent tweets to check using the slider
  6. Click "Start Download"
  7. Your files will be saved in a folder named after the username

Note: Downloading large amounts of content may take time depending on your internet connection.

What formats are available?
  • Video HD: Downloads videos in the highest quality available
  • Video SD: Downloads videos in standard definition (smaller file size)
  • GIF: Downloads animated GIFs in their original format
  • Images: Downloads all images attached to a tweet in full resolution
Where are my downloaded files saved?

All downloads are saved in the "downloads/twitter" folder in the same directory as this application. For user profile media, a subfolder will be created with the username if that option is selected.

Files are named based on tweet ID and content type to ensure they are unique.

Common issues and troubleshooting
  • Rate limiting: Twitter may limit the number of requests you can make. If you see an error about rate limiting, wait 15 minutes before trying again.
  • Protected tweets: You cannot download content from protected/private accounts.
  • No media found: If the user or tweet doesn't contain the type of media you selected, you'll see a "No media found" message.
  • Authentication errors: Some content might require authentication. The downloader uses anonymous access which may limit some downloads.
